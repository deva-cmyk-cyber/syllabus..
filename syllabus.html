SEM II SYLLABUS

Course Overview

This course covers essential front-end web development technologies. Starting with "Basic HTML", you'll learn web page structure. "Advanced HTML" explores semantic HTML and accessibility."CSS" teaches website styling and layout. Finally, "JavaScript" introduces interactivity and dynamic behavior. You'll gain the skills to build functional and visually appealing websites.

  • Basics of HTML
  • Advanced HTML
  • Fundamentals of CSS
  • Introduction to Javascript
